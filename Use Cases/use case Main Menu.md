**Goal in Context** -
As a Manager, I would like to access a menu which will give me access
 to all required Reports in a clear and easily understandable way.
**Scope** -
Company
**Level** -
Secondary Task
**Preconditions** -
All Correct Reports are implemented
**Success Condition** -
Every Report type can be accessed via Menu
**Failed Condition** -
One or More Reports cannot be accessed via Menu
**Primary Actor** -
Manager
**Trigger** -
A request for Any Report is received by Manager
**Main Success Scenario** -
1) A request for any Report is Received by Manager
2) A Manager Navigates the Menu
3) A Manager Finds correct Report in Menu
**Extensions** -
None
**Sub-variations** -
None
**Schedule** -
Due Date: Release 1.0
